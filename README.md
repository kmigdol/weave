# Weave

A 32-bit arcade traffic-weaving game. Thread through dusk Bay Area freeway traffic, chain near-misses, draft for boost.

**Built for the [2026 Vibe Coding Game Jam](https://jam.pieter.com/2026).** Submission deadline: **May 1, 2026 @ 13:37 UTC**.

> 90%+ of the code in this repo is written by AI per jam rules. Creative direction, gameplay tuning, and architectural calls are human.

## Status

In development. See [the design doc](docs/plans/2026-04-09-weave-design.md) for the full spec and [Linear](https://linear.app/nextbest/team/weave) for milestone tracking.

## Stack

- Vite 8 + TypeScript 6
- Three.js (WebGL renderer, custom CRT post-process)
- Tone.js for audio
- Supabase for the global leaderboard
- Vercel for hosting

## Play

Once deployed, the game will be playable at a Vercel URL linked here. No install, no sign-up. Desktop and mobile.
